# **App Name**: MediShelf

## Core Features:

- Medicine Inventory Management: Users can add, view, edit, and remove medicine entries, including details like name, type, quantity, and purchase date.
- Expiry Date Tracking: Functionality to input and monitor expiry dates for each medicine, providing clear visual indicators of their status.
- Expiry Alert System: Provides in-app notifications and visual cues to alert users when medicines are nearing or have passed their expiry dates.
- Search and Filtering: Enables users to efficiently search for specific medicines or filter their inventory based on criteria like name, expiry status, or type.
- Responsive User Interface: Ensures the application provides an optimal and consistent user experience across various devices, including mobile phones, tablets, and desktop web browsers.
- Dark/Light Theme Toggle: Allows users to easily switch between a default dark theme and an alternative light theme for personalized visual comfort.

## Style Guidelines:

- Primary color: A sophisticated, slightly desaturated indigo-violet (#7E57FF) to denote reliability and professionalism, contrasting effectively against dark backgrounds.
- Background color: A very dark, subtle cool gray with hints of the primary hue (#302B3A), providing a deep and rich canvas for the interface, designed to highlight elements in a dark theme.
- Accent color: A bright, clear sky blue (#A2CEFF) for highlights, alerts, and interactive elements, providing high contrast and enhancing the 'liquid glass' effect.
- Body and headline font: 'Inter' (sans-serif) for its modern, neutral, and highly legible qualities, ensuring clarity and professionalism for all text elements.
- Minimalistic and line-based icons, utilizing soft gradients or subtle inner glows. Icons will be crisp and conceptual, consistent with a clean and modern glassmorphic aesthetic.
- A fluid and adaptive modular layout with ample padding. Information will be presented within frosted, semi-transparent card-like elements that feature blurred backgrounds and subtle, diffused shadows to achieve the 'liquidglass glasomorphism'.
- Subtle and fluid animations for transitions, hover states, and data updates. Emphasis on animated blurs, light shifts, and slight parallax to enhance the 'liquid glass' feel and provide a premium user experience.
